Hotel Management System
Python microproject for hotel management

Description
This project is a simple Python-based hotel management system. It simulates basic hotel operations by allowing users to book rooms, check availability, manage customer details, and generate bills.

Rooms Available
Single Room
Double Room
Deluxe Room

Features
Room booking system
Check room availability
Customer details management
Prevents double booking
Billing system with total cost calculation
Booking confirmation

 Technologies Used
Python 3

 How to Run
Download the file
Open terminal / command prompt
Run:
Bash
python main.py# Hotel-Management-System
